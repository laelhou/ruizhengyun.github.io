# 注释规范
### 模块功能
```javascript
/*-
 - ------------------------------------------------------------------
 - 模块描述说明
 - ------------------------------------------------------------------
 */
```


### 模块内的小函数方法归类
```javascript
/*-
 - 小函数方法归类说明，这些零散的小函数方法放在一起 对应 一个业务方法逻辑
 - ------------------------------------------------------------------
 */
```

/*-
 - 函数功能简述
