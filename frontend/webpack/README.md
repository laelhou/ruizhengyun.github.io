# 前言

要说 Webpack 本身官方已经有详尽文档，社区也有无数文章，笔者将从自己多年学习的经验和自己的角度来重学 webpack 并记录一些笔记，希望对读者有所收益。

其实，早几年搭建 React 等脚手架就用到 webpack，相关知识也林林总总知道一些，其中曲折问题坑也填过很多。近来，老项目要升级，再想想一路走来磕磕绊绊，让我下定决心，系统学习下 webpack。
