# CSS命名

确保文件命名总是以字母开头而不是数字，且字母一律小写，用半角的连词线 ( - ) 分隔且不带其他标点符号

my-address.c[le|sa]ss

# ClassName命名

ClassName的命名应该尽量精短、明确，必须以字母开头命名，且全部字母为小写，用半角的连词线 ( - ) 分隔且不带其他标点符号

# Name 项目名或公司名

name 为 tf、ant、jd、bd、ali
