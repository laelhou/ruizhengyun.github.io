# 网络

# 带宽

通信中在给定范围内的最高频率和最低频率之差。例如，电话线的波段宽度是3000HZ，它是可以传送的最高频率 3300Hz 与最低频率 300Hz 的差。在计算机网络中，带宽越大，数据就传送得越快。数据传送的速度单位是波特率（bps）。

# bps

波特率的英文缩写，它是调制解调器这类数据传送设备的速度度量指标。

# 网桥

将多个网络、子网或环联接成一个一个大逻辑网。网桥中保留节点地址表，在它的基础上，可以向特定的子网转发数据包，这样就减少了在其他子网上的传送。在网桥化的网络中，到任何目标都只有一条路径（否则数据包可能会在网上循环而导致网络堵塞）。网桥比转发器复杂。

# 网桥路由器

它是网桥和路由器的综合。通常情况下，如果只有一个传输协议（例如TCP/IP），网桥路由器和路由器相似，都是向目标发送按详细路由格式化的数据包。而对于其他所有类型的数据包（例如IPX），网桥路由器又与网桥类似，就象这些数据包不是在本地一样，只是将它们从原始位置传送到LAN段。

# 网络路由器

它是将网络连接起来并将网络信息导向到其他网络上的设备，通常网络信息会自动寻找多个路由器，并选择效率最高的路由。

# 子网

网络的一部分，它可以是物理上独立的网络段，与网络的其它部分共享网络地址，并用子网号区分。子网与网络的关系和网络与网际的关系相同。

# 子网掩码

一个32位的值，它使IP报文的收件人能区分IP地址的网络ID部分和主机ID部分。

# 网卡

用于计算机连接到局域网的扩展卡或其它设备。也称作网络网络适配器、网络适配卡、适配卡或网络接口卡（NIC）。

# 网卡驱动程序

它是直接作用于网络适配卡的网络设备驱动程序，是卡与协议驱动程序之间的中介。通过Services forMacintosh，服务器上的APPleTalk Protocol将作为协议驱动程序，并绑定到一个或多个网络适配卡驱动程序上。

# 默认所有者

当删除文件夹的账号或卷的前任所有者时，或文件夹的账号与卷的前任所有者之一到期时，将由默认所有者获取服务器上的文件夹的所有权。每个服务器有一个默认所有者，也可指定所有者。

# DHCP

动态主机配置协议的缩写，它承担IP地址和相应信息的动态地址配置。DHCP提供安全、可靠而且简单的TCP／IP网络配置，避免地址冲突，并且通过地址分配的集中管理帮助保存对IP地址的使用。

# Internet名称服务器（WINS）

在路由选择环境中将WindOWS联网计算机名解析为IP地址的名称解析服务。WINS服务器处理名称注册、查询和释放。

# 多宿主计算机

它是有多个网卡的系统，或是为单个网络接口卡配置了多个IP地址的系统。

# 多协议路由选择

通过连接局域网或将局域网连接到广域网，实现IP和IPX网上的路由选择。

# 名称解析服务器

TCP/IP内部网络要求使用名称解析服务来将计算机名转换为IP地址或将IP地址转换为计算机名。用户用“萧易”名称连接到计算机，程序是使用IP地址。

# 优先权等级

它是给用户账号制定的三个设置之一（User、Admlnistrator、Guest）。用户账号具有的优先等级决定用户可以在网上完成的动作。

# 递归

它是 DNS 名称解释中的三个关键概念之一。解析器将递归解析请求传送到它的本地名称服务器，通知名称服务器解析需要该查询的完整的应答，而不是指向其它名称服务器的指针。速归解析有效地将一些工作放在名称服务器上，这样就使解析服务器更小而且更简单。

# 重建

客户计算机需要周期地更新它们在WINS服务器的 NetBIOS 名称注册。当客户计算机第一次注册到WINS服务器时，该服务器将返回一条消息，指示客户计算机需要在何时更新其注册。

# SMIP

简单邮件传送协议的缩写，是TCP/IP协议的一部分，控制在消息传送代理之间交换电子邮件。

# UDP

用户数据包协议，它作为TCP的补充，提供无连接数据包服务，保证既不传递也不校正已传递的数据包序列。

# FTP

文件传输协议，一种支持本地计算机与远程系统（该协议支持的）之间进行文件传输的服务。
